# Tian-Tech 项目集合

## 项目概述

Tian-Tech 是一个由 Tian-Tech 团队自主开发的综合性技术平台，旨在整合多种现代技术应用，为用户提供一站式的智能服务体验。本平台涵盖了AI助手、实时聊天系统、办公工具、铁路信息查询、用户认证等多个领域，通过模块化设计实现了功能的灵活组合与扩展。

项目始于2024年，最初是为了满足团队内部的技术需求而开发，随着功能的不断完善和扩展，逐渐形成了一个功能丰富的综合性平台。本仓库包含了项目的前端页面、后端服务脚本、API接口和相关资源文件，完整呈现了平台的技术实现和功能架构。

Tian-Tech 平台的设计理念是"技术赋能，简化生活"，通过整合先进的AI技术和现代Web开发技术，为用户提供直观、高效、智能的服务体验。平台不仅是一个实用工具集合，也是团队技术能力的展示窗口。

## 核心子项目详解

### 1. WanQian AI助手

**WanQian** 是Tian-Tech平台的核心项目之一，基于智谱AI ChatGLM API开发的智能助手系统，提供智能聊天和图像生成功能。

- **主要文件**：
  - `WanQianchat.html` - 主聊天界面，支持多模型切换和流式响应
  - `WanQianimagine.html` - 图像生成界面，提供多种风格和参数设置
  - `WanQian.py` - Python版AI助手，支持本地运行和气泡式聊天界面
  - `WanQiannewchat.js` - 聊天核心功能脚本，处理消息发送和接收
  - `WanQianChatimagine.js` - 图像生成功能脚本，处理图像生成请求
  - `WanQianchat.css` - 聊天界面样式文件，实现现代化的气泡式设计
  - `WanQianX.css` - WanQian扩展样式文件
  - `WanQianX.html` - WanQian扩展功能页面

- **技术特点**：
  - **多模型支持**：集成了7种不同的模型，包括WanQianW1-exp2、WanQianCora、WanQianTurbo等，满足不同场景的需求
  - **流式响应**：采用Server-Sent Events技术，实现实时的打字效果，提升用户体验
  - **思考过程显示**：支持显示AI的思考过程，增强用户对AI决策的理解，提高交互透明度
  - **气泡式界面**：采用现代化的聊天气泡设计，区分用户和AI消息，提供流畅的对话体验
  - **本地运行**：Python版本支持完全本地运行，无需依赖Web服务器，适合离线使用场景
  - **Markdown支持**：内置Markdown解析，支持格式化文本显示，包括代码块、列表、链接等
  - **图像生成**：支持根据文本描述生成高质量图像，提供多种风格选择

### 2. LightningChat 聊天系统

**LightningChat** 是一个实时聊天系统，支持创建和加入聊天房间，实现多用户实时交流。

- **主要文件**：
  - `LightningChat.php` - 主聊天服务，处理房间管理和消息转发
  - `chat_api.php` - 聊天API接口，提供消息发送和接收功能
  - `chat_server_web.php` - Web端聊天服务器
  - `check_room.php` - 房间状态检查
  - `create_room.php` - 房间创建服务
  - `LightningChatHall.html` - 聊天大厅，显示可用的聊天房间
  - `LightningChatcreate.html` - 房间创建页面，允许用户创建新的聊天房间
  - `LightningChatjoin.html` - 房间加入页面，允许用户通过房间ID加入聊天

- **技术特点**：
  - **实时通信**：采用轮询机制实现准实时消息传递，平衡性能和可靠性
  - **房间管理**：支持创建、加入和管理多个聊天房间，每个房间独立运行
  - **用户友好**：简洁直观的界面设计，易于使用，适合各年龄段用户
  - **跨浏览器兼容**：支持主流浏览器，确保广泛的设备兼容性
  - **消息历史**：自动保存聊天历史，用户可以查看之前的对话内容

### 3. 登录与认证系统

提供用户注册、登录和个人信息管理功能，是平台其他需要用户身份的服务的基础。

- **主要文件**：
  - `login/index.html` - 登录界面，支持账号密码登录
  - `login/register.html` - 注册界面，支持新用户注册
  - `login/auth.php` - 认证服务，处理用户登录验证和会话管理
  - `login/personal.html` - 个人中心，显示和管理用户个人信息
  - `login/personal.php` - 个人信息管理接口，处理信息更新和保存
  - `login/styles.css` - 登录系统样式文件，实现现代化的表单设计
  - `login/data.json` - 用户数据存储文件

- **技术特点**：
  - **安全认证**：采用会话管理和密码加密，确保用户信息安全
  - **模块化设计**：认证功能可被其他系统轻松集成，提供统一的用户体验
  - **响应式设计**：适配不同屏幕尺寸，提供良好的移动端体验
  - **数据持久化**：用户信息存储在JSON文件中，确保数据安全和可移植性
  - **个人中心**：提供用户个人信息管理功能，增强用户粘性

### 4. 铁路信息系统

提供铁路相关新闻、调图信息和列车时刻表查询功能，为铁路爱好者和出行用户提供便捷的信息服务。

- **主要文件**：
  - `railway/index.html` - 铁路信息主页，展示最新铁路新闻和资讯
  - `railway/train-time.html` - 列车时刻表查询界面
  - `railway/train-time.php` - 时刻表查询后端服务
  - `railway/canshu.html` - 铁路相关参数信息
  - `railway/news/` - 铁路新闻目录，包含多篇铁路相关新闻文章
  - `railway/2026.1.26中国铁路一季度调图.htm` - 2026年一季度铁路调图详细信息
  - `railway/2026.1.26中国铁路一季度调图.files/` - 调图相关资源文件

- **技术特点**：
  - **信息整合**：整合多种铁路相关信息，提供一站式查询服务
  - **实时数据**：定期更新铁路调图和时刻表信息，确保数据准确性
  - **用户友好**：直观的查询界面，便于用户快速获取所需信息
  - **多媒体支持**：支持图片和表格等多种信息形式，丰富内容展示
  - **新闻动态**：提供最新铁路新闻和技术动态，保持内容时效性

### 5. 办公工具套件

提供文档创建、查看和管理功能，为用户提供轻量级的办公解决方案。

- **主要文件**：
  - `office/index.html` - 办公工具主页，展示可用的办公功能
  - `office/create_word.html` - 文档创建界面，支持基本的文本编辑
  - `office/view_word.html` - 文档查看界面，支持多种格式的文档预览
  - `office/file_open_word.html` - 文件打开界面
  - `office/styles.css` - 办公工具样式文件，实现专业的办公界面设计

- **技术特点**：
  - **轻量级设计**：无需安装厚重的办公软件，直接在浏览器中使用
  - **基本编辑功能**：支持文本格式化、列表、链接等基本编辑功能
  - **多格式支持**：支持查看多种常见文档格式，提升兼容性
  - **响应式设计**：适配不同设备，支持移动办公场景
  - **文件管理**：提供基本的文件管理功能，方便用户组织文档

## 技术架构与实现

### 前端技术

- **HTML5/CSS3**：构建现代化的用户界面，支持语义化标签和新特性
- **JavaScript**：实现交互功能和动态效果，处理用户输入和响应
- **Bootstrap**：使用响应式框架，确保跨设备兼容性和一致的设计风格
- **原生JavaScript**：核心功能使用原生JS实现，减少依赖，提高性能
- **CSS3动画**：添加平滑的过渡和动画效果，提升用户体验
- **Canvas**：用于实现复杂的图形效果和动画

### 后端技术

- **PHP**：处理服务器端逻辑，提供API接口，支持动态内容生成
- **Python**：实现本地运行的AI助手和数据处理功能，利用其强大的库生态
- **JSON**：轻量级数据交换格式，用于存储配置和用户数据，便于跨平台使用
- **文件系统**：使用本地文件系统存储数据，简化部署，降低依赖

### API集成

- **智谱AI ChatGLM API**：提供AI聊天和图像生成能力，是WanQian助手的核心

### 存储方案

- **JSON文件**：存储用户数据、配置信息和系统设置，结构清晰，易于维护
- **本地文件系统**：存储上传文件、图片和静态资源，便于直接访问
- **PHP会话**：管理用户登录状态和临时数据，确保会话安全

## 快速开始指南

### 运行 WanQian AI助手

**Web版**：
1. 直接在浏览器中打开 `WanQian/WanQianchat.html`
2. 在顶部模型选择下拉菜单中选择所需的AI模型
3. 在底部输入框中输入问题，按Enter键发送
4. 查看AI的回复，包括可能的思考过程

**本地版**：
目前已经开发了TianTech WanQian AI助手本地Windows版本。可在仓库内获取。

### 启动 LightningChat 聊天系统

1. 确保本地安装了PHP环境（如XAMPP、WAMP等）
2. 将项目部署到Web服务器根目录
3. 在浏览器中打开 `LightningChat/LightningChatHall.html`
4. 点击"创建新房间"按钮创建聊天房间，或输入房间ID加入现有房间
5. 开始与其他用户实时聊天

### 访问其他功能

直接在浏览器中打开相应的HTML文件即可访问其他功能：

- `index.html` - 平台主页
- `login/index.html` - 用户登录
- `railway/index.html` - 铁路信息
- `office/index.html` - 办公工具
- `notice/index.php` - 通知系统

## 注意事项

- **后端依赖**：部分功能（如聊天系统、用户认证）需要PHP环境支持，建议使用XAMPP或WAMP等集成环境
- **API密钥**：AI助手功能需要有效的智谱AI API密钥，确保在使用前配置正确的密钥
- **网络连接**：大部分功能需要网络连接才能正常使用，特别是AI助手和聊天系统
- **浏览器兼容**：推荐使用Chrome、Firefox、Edge等现代浏览器，以获得最佳体验
- **本地运行**：Python版AI助手可在无网络环境下运行基本功能，但AI响应需要网络连接
- **文件权限**：确保PHP文件具有适当的执行权限，特别是处理文件上传和写入的脚本
- **安全配置**：在生产环境中，建议加强安全配置，如设置适当的文件权限、使用HTTPS等
